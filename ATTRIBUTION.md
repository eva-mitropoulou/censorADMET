# Data Attribution

## ChEMBL-Derived Benchmark Input

`data/measurement_records.parquet` is a curated research adaptation of ChEMBL release 36. It includes transformed censoring intervals, benchmark filtering fields, and fixed identifiers required to reproduce the CensorADMET analyses.

The source database is ChEMBL, provided by EMBL-EBI under the Creative Commons Attribution-ShareAlike 3.0 Unported license. This derived dataset is distributed under the same [CC BY-SA 3.0 license](LICENSE_DATA).

When reusing the benchmark input, retain this attribution and cite:

> Zdrazil, B. et al. The ChEMBL Database in 2023: a drug discovery platform spanning multiple bioactivity data types and time periods. *Nucleic Acids Research* 52, D1180-D1192 (2024). https://doi.org/10.1093/nar/gkad1004

Also cite the accompanying CensorADMET preprint as described in [CITATION.cff](CITATION.cff).
