# Changelog

## 1.0.0 — 2026-07-17

First frozen manuscript research-package release.

- Implements the constrained probability-deficit objective and its augmented-Lagrangian optimiser.
- Archives fixed curated ChEMBL 36-derived cohorts, split assignments, configurations, result matrices, figure/table scripts, and verification tooling.
- Includes exact-only, plain Tobit interval-likelihood, weighted-Tobit, soft-deficit, AFT, calibration, and semi-synthetic analysis artifacts.
- Scope: released artifacts audit the submitted manuscript; full model-grid retraining is intentionally not run by the lightweight validation workflow.
